```
show user; 显示当前用户名

Select 查询单个表中的数据
Order By 按升序或降序对查询的结果集进行排序
Distinct 消除查询输出中的重复行
Where 为查询返回结果集中的行记录指定过滤条件
And 组合两个或两个以上的布尔表达式，如果所有表达式都为true，则返回true
Or 组合两个或两个以上的布尔表达式，如果其中一个表达式为true，则返回true
Fetch 使用行限制子句限制查询返回的行数
in 使用行限制子句限制查询返回的行数
Between 基于一系列值(区间值)过滤数据
Like 根据特定模式执行匹配
Inner join 从表中查询具有与其他表匹配的行记录
Left join 左连接概念，并学习使用它选择左表中具有，但右表中不具有的行记录
Right join 解释右连接概念，并从右表查询具有，但左表中不具有的行记录
Cross join –从多个表中构建笛卡尔乘积
Self join 将表连接到自身以查询分层数据或比较同一个表中的行记录
Group By 将行分组为子组，并为每个分组应用聚合函数
Having 过滤分组中的行记录
子查询 - 子查询的概念以及使用子查询来执行高级数据选择技术
相关子查询 - 相关的子查询，它是一个依赖于外部查询返回的值的子查询
EXISTS和NOT EXISTS - 检查子查询返回的行是否存在
ANY，SOME和ALL - 将值与列表或子查询进行比较
UNION - 将两个查询的结果合并为一个结果
 INTERSECT - 实现两个独立查询的结果的交集
MINUS - 学习从一个结果集中减去另一个结果(也就是求差集)
 INSERT - 学习在表中插入一行
INSERT INTO SELECT - 从查询结果中将数据插入到表中
INSERT ALL - 多重插入语句，将多行插入到一个或多个表中
UPDATE - 更新表的存在的数据值
 DELETE - 从表中删除一行或多行
MERGE - 使用单个语句逐步完成插入，更新和删除操作
CREATE TABLE - 在数据库中创建新表
IDENTITY列 - 使用IDENTITY子句来定义表的标识列
ALTER TABLE - 改变表的结构
ALTER TABLE ADD列 - 显示将一个或多个列添加到现有表
ALTER TABLE MODIFY列 - 更改表中现有列的定义
DROP COLUMN - 使用各种语句从表中删除一列或多列
DROP TABLE - 从数据库中删除表
TRUNCATE TABLE - 更快，更有效地删除表中的所有数据
RENAME TABLE - 学习重命名表和处理其依赖对象的过程
Oracle数据类型 - 内置Oracle数据类型的概述
NUMBER - 数字数据类型，并展示使用它为表定义数字列
FLOAT - 通过实例来解释Oracle中的浮点数据类型
CHAR - 固定长度的字符串类型
NCHAR - 存储固定长度的Unicode字符数据，并解释CHAR和NCHAR数据类型之间的区别
VARCHAR2 - 向您可变长度字符，并向您展示在表中定义可变长度字符列
 NVARCHAR2 - 在数据库中存储可变长度的Unicode字符
DATE - 日期和时间数据类型，并说明有效地处理日期时间数据
TIMESTAMP - 以小数秒精度存储日期和时间
INTERVAL - 区间数据类型，主要用来存储时间段
TIMESTAMP WITH TIME ZONE - 使用时区数据存储日期时间
主键 - 解释主键概念，并使用主键约束来管理表的主键
外键 - 解释外键概念，并使用外键约束来强制表之间的关系
NOT NULL约束 - 确保列不接受NULL值
UNIQUE约束 - 确保存储在一列或一组列中的数据在整个表内的行之间是唯一的
CHECK约束 - 在将数据存储到表中之前添加用于检查数据的逻辑的过程
```
